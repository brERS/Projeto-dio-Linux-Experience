O que são Clients?

INSTRUÇÕES:
Cliente é um termo empregado em computação e representa uma entidade que consome os serviços de uma outra entidade servidora,
em geral através do uso de uma rede de computadores numa arquitetura cliente-servidor.
Saiba mais em Wikipedia

O que são Servers?

INSTRUÇÕES:
Servers em informática é um software ou computador, com sistema de computação centralizada que fornece
serviços a uma rede de computadores, chamada de cliente.
Saiba mais em Wikipedia

Entendendo a diferença entre listas ordenadas e não ordenadas

INSTRUÇÕES:

As listas não ordenadas ou numeradas são usadas para listar itens, sem se preocupar com sua sequência. Chamamos de lista de marcadores apenas.

As listas ordenadas ou numeradas são usadas para indicar alguma sequência ou numeração
